# Product Roadmap (v2+)

## Near-term

- Queue-backed ingestion and chunked uploads for multi-GB batches
- Provider adapters for Vertex AI / Bedrock / Azure OpenAI runtime switching
- Rich visualization exports (PDF report, XLSX workbook, Notebook package)

## Mid-term

- Dataset workspace with schema mapping
- Scheduled re-analysis and anomaly alerting
- Collaboration and role-based team spaces

## Long-term

- Financial benchmarking against sector baselines
- Explainable AI trace for each generated insight
- Fully managed enterprise deployment profile
